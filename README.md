# Simple SCSS and JS compiler base project
1. Clone project by tying `git clone this_repo_link`
2. Install dependencies by typing `npm install`
3. Init your project by typing `npm run build`
4. Type `npm run start` to start the watcher whichs updates the static dist files when you make changes to the src files
5. Optional: install browser-sync and point it to run from the dist folder so you have a watcher and live browser reload at once
